## How to Add New Projects

1. Create a new folder in the root directory
2. Add your `index.html` file to the folder
3. Add other project files (CSS, JS, etc.) as needed
4. Update this README.md with a link to your new project
5. Push to GitHub and wait 2-3 minutes for GitHub Pages to rebuild
